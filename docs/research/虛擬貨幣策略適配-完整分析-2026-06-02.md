# 虛擬貨幣到底適合什麼策略 —— 用我們自己跑過的證據

> 2026-06-02。本文不是教科書空話,每個結論都綁這個專案實際跑過的 walk-forward OOS、
> pool t-stat、bootstrap 95% CI、含 2022 熊市的回測,以及今天剛跑的時間效應 probe。
> 立場:嚴格中立、優先講「什麼會賠」。

---

## 0. 一句話總結

**Crypto 的 edge 在「順勢動量 + 衍生品結構性失衡(funding / OI / 散戶持倉)」,
不在「均值回歸 / 日曆時鐘季節性 / 幾何形態」。**
而且真正殺帳戶的不是選錯策略,是**有效槓桿 + 強平**。

配方一句話:**long-biased 趨勢/動量為核心 + 一小條市場中性的結構性 alpha sleeve + 少量諧波 PRZ 去相關,固定倉位、有效槓桿壓在 ~1–3×。**

---

## 1. 為什麼可以這樣斷言(證據基礎)

- 策略池 20+ 條、20 個幣、walk-forward 樣本外(OOS)、pool t-stat 為主(非全期報酬)。
- 多重檢定防灌水:Deflated Sharpe + Bonferroni + BH-FDR;顯著還要過「magnitude 雙過濾」。
- 含 **2022 熊市**深歷史(非平穩性壓力測試);多空引擎含強平。
- 進策略前的三重過濾:**跨幣方向一致 + 經濟意義清晰 + 跟 price corr < 0.3**。

這套紀律本身就是結論的一部分:**大多數「看起來會賺」的東西,過了這關就死了。**

## 2. Crypto 的「性格」——決定了什麼能賺

### 2.1 強趨勢 / 動量 = 王(long-biased)
- **驗過會賺**:`tsmom`、`dual_mom_ls`、Donchian/breakout 家族、`macd`、`rsi_stoch`、`price_action`。
- **數據量級**:動量核心年化 ~24%、但 maxDD ~60%(crypto 的代價)。
- **為什麼 crypto 特別吃這套**:24/7 無休、強反身性(漲→FOMO→更漲)、散戶主導、市場相對低效、報酬肥尾。趨勢一旦成形拖很久。
- **方向偏多**:crypto 長期上行 + 借券做空成本/強平風險高 → 純多空對稱在這裡常常**做空腿拖累**(見 3.1)。

### 2.2 衍生品結構性失衡 = 唯一真正「去相關」的 alpha
這是 crypto **獨有**、且最值得投資的方向(永續合約特性):
- **Funding rate**:`funding_extreme` / `funding_momentum` — funding 與 price 弱相關(probe t=5.93)、是真結構性訊號。極端 funding = 槓桿擁擠度。
- **OI / 散戶持倉(retail_ls)**:`oi_contrarian`(翻成 contrarian 後)、`retail_ls` / Delta 版(pool t=+3.30、Q2 唯一 95% CI 顯著、shadow 首發 SHORT picks)。
- **為什麼珍貴**:這些訊號**跟價格弱相關** → 加進組合的去相關價值高(不是換個方式再看一次價格)。這是組合 Sharpe 的真正來源。
- **反例(別自欺)**:`volume` 當結構訊號 t≈0(跟 price 高相關,沒有獨立資訊)。**新「極端 X」策略前先驗 corr(X, price)。**

### 2.3 諧波 PRZ = 低估的小回撤冠軍(但「機械諧波」已蓋棺)
- **機械諧波**(純抓 XABCD/Gartley/Bat 比例自動進場)= **無 OOS edge,蓋棺**。
- **PRZ 匯流版**(`harm_prz_*`:把諧波 PRZ 當「支撐/壓力匯流區」而非機械訊號)= 熊市審計裡的**低估冠軍**,回撤小、跟動量去相關。
- 差別就是:**諧波當「區域共識」有用,當「機械訊號」沒用。** 你的 fib_confluence 也是同個道理(分級點位+連2根站穩,而非單點接刀)。

### 2.4 波動度突破 / squeeze
- `SqueezeBreakout` / `VolatilityBreakout` —— crypto 波動聚集(vol clustering)明顯,壓縮後擴張可交易。
- 屬於「regime 轉換」類、跟趨勢動量有部分重疊但 timing 不同,可當輔助。

## 3. Crypto 的「陷阱」——驗過會賠 / 沒 edge 的(重點看這節)

### 3.1 裸均值回歸(逆勢 MR)→ 爆倉
- **ETF/股市的王(MR)在 crypto 是死刑。** 原生逆勢多空 MR(賭超漲超跌回歸)**會爆倉** —— 趨勢拖太久,逆勢腿在強平前等不到回歸。
- 同一支 `bollinger`:crypto −10%、ETF +8% —— **同策略跨市場命運相反**。
- MR 在 crypto 只能當「趨勢中的回撤進場」(順勢回踩),不能當「對抗趨勢的反轉」。

### 3.2 日曆 / 時鐘季節性 → 無 OOS(今天剛 probe 證實)
- `tools/time-probe.py`:hour-of-day in-sample 超猛(hour-19 t=7.66、20/20 全同號),
  **但前後半樣本 corr = 0.002 → 純假象、OOS 蒸發。**
- Session 區塊量級僅 1–1.5bps(< 來回成本);funding **結算 timing** t<1.4 無效。
- Day-of-week in-sample 大但 crypto 週效應出名非平穩(機構/ETF 進場後翻轉)。
- **結論:crypto edge 不在日曆/時鐘。** TradingView 的「時間週期/正弦線/斐波時區」= numerology。

### 3.3 幾何形態 numerology(Gann / Elliott / Fib 圈螺旋弧扇)→ 無證據
- **江恩**:占星等級,時間-價格「正方」比例任意,無可信 OOS。
- **艾略特波浪**:主觀 + 事後諸葛,機械化不灌 lookahead 極難,文獻 OOS 差。
- **Fib 圈/螺旋/速度阻力弧/扇/楔形**:任意錨定。**你已踩過坑** —— fib 0.886 錨定 pivot 直接把 base edge 毀掉(53%→−4%)。圈/螺旋是同病更重版。

### 3.4 描述性統計當處方 filter
- 觀察「X 在 Y 條件下強」≠ 強制只在 Y 開倉就會復現。
- BTC regime filter 案例:`widepz_btcSide` −0.99(處方化後更差)、只有 `tsmom_btcNotUp` +0.16 真實。

### 3.5 過度優化的 sizing overlay = 偽裝的槓桿
- 改善 Sharpe 的 sizing overlay 要 **cap scalar 到 1.0 再測一次**;gain 消失 = 靠槓桿不是擇時。
- Q1 三項(conf-sizing / conf 校準 / vol-targeting)全驗完**不部署**,固定倉位才是對的。

### 3.6 regime overlay 用自身權益當訊號 = 自我參照灌水
- 用策略自身權益路徑當 regime 訊號會假象灌水(自身 SMA ×2.5 假、BTC 外生 ×1.5 現形)。
- 任何 timing/縮倉 overlay 的 regime 訊號**必須外生**。

## 4. 跨市場對照(為什麼 crypto 特別)

| 市場 | 王道策略 | B&H 難度 | 主動空間 |
|---|---|---|---|
| **Crypto** | 動量 / funding 結構性 | B&H 很強(長期上行) | 動量空間大、但要贏 B&H 難;**去相關 sleeve 才是加值** |
| **ETF(SPY 等)** | 均值回歸 | B&H 難贏(Sharpe ~0.63) | 小 |
| **美股個股** | 動量 + MR 都行 | 中間 | 中 |

統一律:**B&H 越爛、主動空間越大。** Crypto 的 B&H 很強 → 主動策略門檻高 → 真正的價值在「**跟 B&H 去相關**」(動量在跌段保護、結構性 alpha 市場中性),不是單純想跑贏。

## 5. 真錢層面 crypto 特有的鐵律(這節比選策略重要)

1. **有效槓桿 = 真風險**:殺帳戶的是**名目/權益**,不是交易所槓桿數字。5× 有效槓桿在 crypto 必強平。**名目壓 ~1× 權益**。
2. **回測一律含多空 + 強平**,否則做空腿/爆倉風險被藏起來。
3. **去相關真實,但等權 = 死**:4 條去相關 alpha 等權 → maxDD 267%。防禦鏈:風險加權 → CB + DD-aware(46%→16%)→ 有效槓桿 ~3×(→48% 活)。**plain DD-aware ×2.1 是最佳誠實防禦天花板;3× 別用 step。**
4. **walk-forward 漂亮 ≠ pool t 顯著**:換腿/升 live 前必過 strat-validate pool t-stat(bb_revert×ETH Sharpe 2.00 → t 不顯著,紀律救命)。
5. **加碼壓力下別趕**:降風險(砍集中)可現在做;加風險(上新策略)要等驗證。每月加錢是「更要紀律」、不是「趕快上滿」。

## 6. 結論:Crypto 的策略配方

**核心引擎(吃趨勢)**
- Long-biased 趨勢/動量:tsmom / dual_mom / donchian breakout / macd / price_action。
- 固定倉位、有效槓桿 ~1–3×、含強平的多空驗證。

**去相關 sleeve(吃結構性失衡、市場中性)**
- Funding extreme/momentum、OI/retail_ls Delta contrarian、橫截面動量(market-neutral)。
- 鐵律:任何新「極端 X」策略,先驗 **corr(X, price) < 0.3**。

**衛星(去相關小回撤)**
- 諧波 **PRZ 匯流**(非機械諧波)、fib_confluence(分級+站穩、非單點接刀)。

**風控(比選股重要)**
- 組合風險加權 + CB + DD-aware;有效槓桿封頂;cash 緩衝;新錢半倉。

**明確避免**
- 裸逆勢 MR、日曆/時鐘季節性、Gann/Elliott/Fib 幾何 numerology、機械諧波、
  描述性 stats 當 filter、過擬合 sizing overlay、自我參照 regime overlay。

**下一個真 alpha 邊界(roadmap)**
- 不是回頭堆價格形態,而是 **Q3 microstructure**(訂單流 / 盤口 / 大單偵測)——
  那是還沒被你榨過、且天然跟價格序列去相關的資訊源。
