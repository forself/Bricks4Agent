# 遠端管理介面 (Remote Management Interface)

## 概述

遠端管理介面是一個跨平台的管理系統，提供 Web 和 App 版本，
用於管理 Bricks4Agent 的所有安全模組、審核系統和系統配置。

### 即時通訊

使用 WebSocket 進行即時更新：

```typescript
// 事件類型
type WSEvent =
  | { type: 'login'; data: LoginEvent }
  | { type: 'lock'; data: LockEvent }
  | { type: 'alert'; data: AlertEvent }
  | { type: 'review'; data: ReviewEvent };

// 訂閱頻道
ws.subscribe('security:*');      // 所有安全事件
ws.subscribe('review:pending');  // 待審核通知
ws.subscribe('alert:critical');  // 嚴重警示
```

## 安全考量

### 認證
- JWT + Refresh Token
- 強制 MFA (管理介面)
- IP 白名單 (可選)
- Session 管理

### 授權
- RBAC 角色權限
- 資源級權限
- 操作審計

### 資料保護
- HTTPS Only
- 敏感資料加密
- 日誌脫敏
- CORS 設定
