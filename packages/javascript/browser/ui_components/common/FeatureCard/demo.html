FeatureCard 功能展示卡片

用於展示產品功能、Demo 或特性的精美卡片元件

📦 基本用法

new FeatureCard({
    title: 'Action Button',
    description: '流程操作按鈕 - 支援 8 種操作類型，包含 CRUD 與審核流程',
    tags: ['按鈕', '流程', 'CRUD'],
    url: '#'
}).mount('#container');

🏷️ 徽章樣式

new FeatureCard({
    title: 'Advanced Charts',
    badge: 'PRO',
    badgeColor: '#ff6b6b',
    description: '進階圖表元件 - 包含 7 種專業關係圖表',
    tags: ['圖表', '視覺化']
}).mount('#container');

🖱️ 點擊事件與自訂資料

new FeatureCard({
    title: 'Interactive Demo',
    description: '點擊我查看詳細資訊',
    tags: ['互動', '事件'],
    customData: { id: 123, type: 'demo' },
    onClick: (data) => {
        ModalPanel.alert({ message: `點擊了卡片！ID: ${data.id}, Type: ${data.type}` });
    }
}).mount('#container');

📐 無 Hover 上升效果

new FeatureCard({
    title: 'Static Card',
    description: '這張卡片沒有 hover 上升效果',
    tags: ['靜態'],
    elevated: false
}).mount('#container');

☀️ 淺色模式

const card = new FeatureCard({
    title: 'Light Mode Card',
    description: '適用於淺色背景的卡片樣式',
    tags: ['淺色', '主題']
});
card.setLightMode(true);
card.mount('#container');
