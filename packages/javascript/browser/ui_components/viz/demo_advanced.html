Advanced Visualization Components (v2)

互動增強版：拖放功能、點擊細節卡片、巢狀組織圖

1. 關聯圖 (Relationship Network)

[互動]：嘗試拖曳節點改變佈局。滑鼠懸停查看詳細卡片。

2. 組織圖 (Organization Structure)

[互動]：點擊"+"圓點展開/收合。點擊卡片查看人員詳情。

3. 單位階層圖 (Unit Hierarchy)

[互動]：點擊單位查看詳情 (巢狀組織圖)。可多層堆疊詳細卡片。

4. 時序泳道圖 (Event Timeline)

[互動]：滑鼠懸停事件查看詳細資訊。

5. 流程桑基圖 (Process Flow)

[互動]：滑鼠懸停路徑/節點查看流量細節。

6. 階層火焰圖 (Event Hierarchy)

[互動]：查看事件執行堆疊與耗時佔比。

7. 組織旭日圖 (Sunburst View)

[互動]：放射狀查看組織規模分佈。
