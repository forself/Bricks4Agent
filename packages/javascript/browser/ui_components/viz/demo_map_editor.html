🎨 MapEditor - 圖片編輯器

上傳地圖或圖片，添加文字與標註，即時匯出結果

📖 使用說明

  • 上傳圖片：點擊「📁 Upload Image」按鈕選擇圖片
  • 添加文字：選擇「📝 Text」工具，點擊畫布輸入文字
  • 繪製形狀：使用矩形、圓形、線條、箭頭工具，拖曳繪製
  • 編輯元素：使用「↖️ Select」工具選擇並拖動元素
  • 修改文字：雙擊文字元素即可編輯內容
  • 刪除元素：選中後點擊「🗑️ Delete」或按 Delete 鍵
  • 匯出 PNG：點擊「💾 Export PNG」下載圖片（含座標、文字等元數據）
  • 載入 PNG：匯出的 PNG 可再次載入，自動還原所有標註

⌨️ 快捷鍵

Delete 刪除選中元素 • Ctrl+Z 復原 • 雙擊 編輯文字

✨ 功能特點

🖼️ 圖片上傳

支援上傳任意圖片作為底圖，自動適應畫布大小

📝 文字標註

自訂字體大小與顏色，拖曳調整位置，雙擊編輯內容

🎨 形狀工具

矩形、圓形、線條、箭頭，可設定顏色與粗細

🔄 互動編輯

選擇、拖動、刪除元素，支援復原操作

💾 多格式匯出

匯出 PNG 圖片或 JSON 配置檔，隨時保存與載入

⚡ 零依賴

純 JavaScript + Canvas API，無需任何第三方函式庫
